  				Настройка и начало работы с Git

1. скачивание и установка Git на РС с сайта https://git-for-windows.github.io/
2. первоначальная настройка: аутентификиция/идентификация пользователя работающего с СКВ
	1) git config //утилита для указания настроек
	2) git config --global user:name "Имя пользователя"
	3) git config --global user:email email_пользователя
        4) чтобы проверить текущие настройки нужно вызвать утилиту git config и ключ к необходимому параметру:
	git config user:name
3. настройка локального репозитория:
	1) кликаем правой кнопкой мыши на репозиторий которому хотим дать статус master и выбираем Git Bush Here
	2) набираем git init// создание репозитория готового работать с Git  с минимально необходимыми настройками
4. регистрируемся на https://github.com/ - имейте ввиду,что email должны совпадать + подтверждаем регистрацию
5. создаем новый репозиторий: Create new repository
	1) назначаем ему имя, если мы этого не сделаем то система присвоит дефолтное origin;
	2) пишем description репозитория//данный шаг можно пропустить, это не обязательное поле
	3) обязательно ставим галочку в чекбокс Initialize this repository with a README!!!
	4) можно добавить gitignore & licence;
6. подключение к удаленному репозиторию:
	1) git remote add [name] [url]// name- имя удаленного репозитория;
                                         url - SSH, копируем с http://prnt.sc/8xe1nv
        2) передача данных будет осуществляться по сетевому протоколу Secure Shell, для того чтобы его использовать нужно сгенерить ключи
	3) генерация ключей описана https://help.github.com/articles/generating-ssh-keys/
		- проверяем нет ли у нас сгенерированных ключей командой: ls -al ~/.ssh
		- если ключей нет то генерируем: ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
		- нажимаем enter  тем асмым сохраняем ключи куда нам предложила система
		- вводим пароль !!!как вводится пароль не видно, так как это линуксовая фича!!!
		- повторно вводим пароль
		- публичный ключ сгенерировался /Users/you/.ssh/id_rsa.pub
	4) необходимо добавить ssh-keys  к своему аккаунту
		- открываем файл с ключем по средствам блокнота: ctrl+a->ctrl+c;
		- добавляем ключ в gitHub
		- галочка в чекбокс!!!обязательно!!!
	5) проверяем соединение: ssh -T git@github.com -> yes ->passphrase: вводите ваш пароль
7. git add .
8. git commit -m "message"
9. git pull server_repository_name master// забрать все данные с удаленного репозитория
10. git push server_repository_name master// положить все данные на удаленный сервак!!!
