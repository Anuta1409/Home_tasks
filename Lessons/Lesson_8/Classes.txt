Внутренние классы - класс который содержится в другом классе.
Есть 4 вида:
1) внутренние;
public class B(){
private int a;
public int b;
	public class A(){
     int c = a + b;//имеют доступ к переменным и методам внешнего класса
	}
}
2) вложенные - сатические. В статическом классе мыможем получать
 доступ только к статическим полям и методам внешнего класса;
public class B(){
private static int a;
public static int b;
	public static class A(){
     int c = a + b;//имеют доступ к переменным и методам внешнего класса
	}
}
3) локальные - описывается внутри каного либо метода. Метод внутри которого находится данный метод может возвращить переменную класса: return new LocalClass();
4)анонимные классы (безымянный)- классы которые находятся внутри метоов и не имют собственного имени:
return new method_name or class_name(){}//имя метода или класа который онбудет реализовывать

Рефлексия - механизм позволяющий узнать информацию о классе.Рефлексия в Java осуществляется с помощью Java Reflection API. Этот интерфейс API состоит из классов пакетов java.lang и java.lang.reflect. С помощью интерфейса Java Reflection API можно делать следующее:
1)Определить класс объекта. 
2)Получить информацию о модификаторах класса, полях, методах, конструкторах и суперклассах.
3)Выяснить, какие константы и методы принадлежат интерфейсу. 
4)Создать экземпляр класса, имя которого неизвестно до момента выполнения программы.
5)Получить и установить значение свойства объекта.
6)Вызвать метод объекта.
7)Создать новый массив, размер и тип компонентов которого неизвестны до момента выполнения программ.
